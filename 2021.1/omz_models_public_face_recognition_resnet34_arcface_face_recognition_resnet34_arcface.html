face-recognition-resnet34-arcface

The original name of the model is LResNet34E-IR,ArcFace@ms1m-refine-v1.

Use Case and High-Level Description

Deep face recognition net with ResNet34 backbone and Arcface loss

Input

Original Model

Image, name: data, shape: 1,3,112,112, format: B,C,H,W, where:

  • B - batch size
  • C - channel
  • H - height
  • W - width

Channel order is RGB.

Converted Model

Image, name: data, shape: 1,3,112,112, format: B,C,H,W, where:

  • B - batch size
  • C - channel
  • H - height
  • W - width

Channel order is BGR.

Output

Original Model

Face embeddings, name: pre_fc1, shape: 1,512, output data format: B,C, where:

  • B - batch size
  • C - row-vector of 512 floating points values, face embeddings

The net outputs on different images are comparable in cosine distance.

Converted Model

Face embeddings, name: pre_fc1, shape: 1,512, output data format: B,C, where:

  • B - batch size
  • C - row-vector of 512 floating points values, face embeddings

The net outputs on different images are comparable in cosine distance.
