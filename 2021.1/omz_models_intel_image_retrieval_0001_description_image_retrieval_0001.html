Inputs

Name: input, shape: [1x3x224x224] — An input image in the format [BxCxHxW], where:

  • B - batch size
  • C - number of channels
  • H - image height
  • W - image width

Expected color order: BGR.

Outputs

One [1x256]-shaped tensor — image embedding vector.
